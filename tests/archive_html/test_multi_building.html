Feature Description

This test validates the new multi-building placement functionality where players can place multiple buildings of the same type without having to reselect from the build menu, as long as they have sufficient resources.

Test Steps

  1. Open the game
  2. Complete the tutorial or skip to get to the main village view
  3. Ensure you have enough resources for multiple buildings (you can use developer tools to add resources if needed)
  4. Click on a building type in the build menu (e.g., House)
  5. Place the building on the village grid
  6. Expected Result: If you still have enough resources, you should:
    • Stay in build mode (crosshair cursor remains)
    • See the building button still highlighted
    • Get a toast notification saying you can build another
    • Be able to immediately place another building
  7. Continue placing buildings until you run out of resources
  8. Expected Result: When you can't afford another building:
    • Build mode automatically exits
    • Cursor returns to normal
    • Building button becomes disabled/unaffordable
  9. Test the Escape key to manually exit build mode while still having resources
  10. Test right-clicking to exit build mode

Developer Testing Commands

If you need to add resources for testing, open the browser console and use:

// Add resources for testing
window.gameState.resources.gold += 1000;
window.gameState.resources.wood += 1000;
window.gameState.resources.stone += 1000;
window.gameState.resources.food += 1000;

// Refresh the UI
window.villageManager.updateBuildingButtonStates();
